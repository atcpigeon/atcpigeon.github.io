CS180: Prokudin-Gorskii Colorization Project

Project Overview

This project is part of CS180: Intro to Computer Vision and Computational Photography. The goal is to colorize the Prokudin-Gorskii photo collection by aligning the red, green, and blue channels of each image.

My Approach

I implemented a two-stage process: first, a naive alignment method using exhaustive search, and then an optimized pyramid-based alignment for high-resolution images. Below are some example results using both methods.

Challenges & Solutions

One of the main challenges was aligning high-resolution images efficiently. The pyramid approach significantly reduced the computational cost while maintaining accuracy. I also had to handle different brightness levels between channels, which I addressed using Normalized Cross-Correlation (NCC) as a matching metric.

Bells & Whistles

For extra credit, I implemented automatic cropping to remove misaligned borders, automatic contrast adjustment, and explored white balancing techniques to enhance image quality.
